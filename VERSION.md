**SARGEN™ Framework v1.0**

Status: Initial Public Release

Release Date: March 2026

---

### Versioning Principle

SARGEN™ versions represent formal framework releases.

A version change occurs only when approved modifications materially affect:

* Governance structure
* Core principles
* Definitions and terminology
* Framework architecture
* Official operational guidance

Minor editorial corrections do not require a version increment.
